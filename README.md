# Forest-Cover-Type-Prediction-
Forest Cover Type Prediction Using Machine Learning
EcoType Forest Cover is a machine learning project that predicts the dominant forest cover type of a geographical area using cartographic variables like elevation, slope, soil type, and wilderness information.
Built with a Random Forest Classifier achieving 96.47% accuracy across 7 forest cover classes, the project includes end-to-end pipeline covering data cleaning, feature engineering, EDA, class imbalance handling, feature selection, model tuning, and a live Streamlit web application for real-time predictions.
